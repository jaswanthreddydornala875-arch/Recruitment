Smart Student Productivity Analyzer

Overview
This project predicts a student's productivity score based on daily habits and provides personalized suggestions.

Key Features
- Predicts productivity using Linear Regression
- Gives smart suggestions based on input
- Helps improve daily performance

Tech Used
- Python
- Pandas
- Scikit-learn

How it Works
1. User inputs daily habits
2. Model predicts productivity score
3. System gives suggestions for improvement
